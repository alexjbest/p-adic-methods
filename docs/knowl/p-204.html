The height function \(h\) extends uniquely to the full Mordell-Weil group \(E(\QQ)\) and satisfies \(h(nQ) = n^2 h(Q)\) for all \(n\in \ZZ\text{,}\) and \(Q\in E(\QQ)\) for \(P,Q \in E(\QQ)\) setting

\begin{equation*} (P,Q) = h(P) + h(Q) - h(P+Q) \end{equation*}

we get a pairing on \(E(\QQ)\text{.}\)
