More generally, for any galois stable quotient \(U\) of \(U_n^\et\) we have a similar diagram \(U^\dR = D_{\cris}(U)\) and corresponding maps

\begin{equation*} j_n^\et, j_n^\dR, \operatorname{loc}_{U,p}, j_n^\et \colon X(\QQ) \to \Sel(U)\text{.} \end{equation*}

We have that \(X(\QQ)\) is contained in

\begin{equation*} X(\QQ_p)_U = j_{U,p}^{\et,-1} (\operatorname{loc}_{U,p}(\Sel(U))) \subseteq X(\QQ_p) \end{equation*}

when

\begin{equation*} U = U_n \end{equation*}

we get

\begin{equation*} X(\QQ_p)_n = X(\QQ_p)_{U_n} \end{equation*}

we have

\begin{equation*} X(\QQ) \subseteq \cdots \subseteq X(\QQ_p )_n \subseteq \cdots \subseteq X(\QQ_p)_1 \subseteq X(\QQ_p)\text{.} \end{equation*}
