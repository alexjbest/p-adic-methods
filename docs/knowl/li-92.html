Item 1.
Let \(D_1, \ldots, D_g \in \Div^0(X)\) representing basis elements of \(J(\QQ)\otimes \QQ\) then compute global height pairings. \(h(D_i ,D_j)\) using B.-Besser-Muller. A basis for spaces of bilinear forms on \(J(\QQ)\) is
\begin{equation*} \frac12(f_kf_l + f_l f_k) \end{equation*}
so compute
\begin{equation*} \frac12(f_k(D_i)f_l(D_j) + f_l(D_i) f_k(D_j)) \end{equation*}
do linear algebra to compute
\begin{equation*} \alpha _{k,l} \end{equation*}
\begin{equation*} h(D_i,D_j) = \sum \alpha _{k,l} ( \frac12(f_k(D_i)f_l(D_j) + f_l(D_i) f_k(D_j))) \end{equation*}
