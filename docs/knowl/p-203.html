There is exactly on odd function

\begin{equation*} \sigma (t) = t + \cdots \in t \ZZ_p\lb \lb t\rb \rb \end{equation*}

and constant \(c\in \ZZ_p\) that together satisfy the \(p\)-adic differential equation

\begin{equation*} x(t) + c = \frac{-\diff}{\omega }\left(\frac 1\sigma \frac {\diff \sigma }{\omega }\right) \end{equation*}

\(\omega \) is the invariant differential associated to a chosen Weierstrass model for \(E\)

\begin{equation*} \omega = \frac{\diff x}{2y + a_1x + a_3} \end{equation*}

and

\begin{equation*} c = \frac{a_1^2 + 4a_2 - E_2(E,\omega )}{12} \end{equation*}

we'll say more about \(E_2(E, \omega )\) in a bit!
