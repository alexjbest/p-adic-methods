Proposition 2.31.

To get \(N\) correct digits in the expansion after reduction we need to start with precision

\begin{equation*} N_1 = N + \max\{ \floor*{ \log_p(2M -3)},\floor*{ \log_p(2g+1)}\} + 1 + \floor*{\log_p(2g-1)}\text{,} \end{equation*}

where \(M\) is the smallest integer s.t.

\begin{equation*} M - \max\{ \floor*{\log_p(2M+1)},\floor*{\log_p(2g+1}\}\text{.} \end{equation*}
