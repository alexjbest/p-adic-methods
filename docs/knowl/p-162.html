  1. Compute a local coordinate at \(P\text{,}\) \((x(t), y(t))\)
  2. For each \(k\text{,}\) write \(\eta_k(x,y)\) as \(\eta_k(t)dt\text{.}\)
  3. Let \(I_{n+1} = 1\text{.}\) Compute for \(k = n, n-1, \ldots, 2\)
    \begin{equation*} I_k = \int_P^{R_{k-1}} \eta_k I_{k+1} = \int_P^{t(R_{k-1})} \eta_k(t) I_{k+1} dt \end{equation*}
    where \(t(R_{k-1})\) is parameterising points in the residue disk of \(P\text{.}\)
  4. \(\int_P^Q \eta_1 \ldots \eta_n = \int_P^{t(Q)} \eta_1(t) I_2(t) dt\text{.}\)
