The defining equations of the gauge transformation \(C_x\) imply that

\begin{equation*} \Res(\Omega _x^\transpose Z\diff \Omega _x - \eta) = 0 \end{equation*}

for all \(x\in D\text{.}\) Since the kernel of

\begin{equation*} H^1_{\dR}(Y_\QQ) \to L^d \end{equation*}

given by the residues at all \(d\) points \(x\in D\) is precisely \(H^1_{\dR}(X)\text{,}\) so the first condition implies that \(\eta\) is unique.
