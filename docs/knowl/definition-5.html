Definition 2.58.

The cyclotomic \(p\)-adic height is a symmetric bilinear pairing

\begin{equation*} h \colon \Div^0(X) \times \Div^0(X) \to \QQ_p \end{equation*}
\begin{equation*} (D_1,D_2)\mapsto h(D_1, D_2) \end{equation*}

disjoint support.

where

  1. \begin{equation*} h(D_1,D_2) = \sum_v h_v(D_1,D_2) = h_p(D_1,D_2) + \sum_{v\ne p} h_v(D_1,D_2) \end{equation*}
    \begin{equation*} =\int_{D_2} \omega _{D_1} + \sum_{v\ne p }m_v \log_p(v) \end{equation*}
    with the left term a coleman integral of the third kind and \(m_v\) some intersection multiplicities.
  2. \begin{equation*} h(D, \div(\beta )) = 0 \end{equation*}
    for \(\beta \in \QQ(X)^\times\text{,}\) so \(h\) is well defined on \(J \times J\text{.}\)
