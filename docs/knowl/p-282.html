A filtered module over a commutative ring #][m R] is an \(R\)-module \(M\) with a collection

\begin{equation*} \{ G^i M \} \end{equation*}

of submodules that is decreasing

\begin{equation*} F^{i+1 } M \subseteq F^i M,\, \forall i\text{.} \end{equation*}

If \(\bigcup F^i M = M\) the filtration is exhaustive. For any filtered \(R\)-module \(M\) the associated graded module

\begin{equation*} \gr(M) = \bigoplus _i \gr^i(M),\,\gr^i(M) = F^iM/F{i+1}M\text{.} \end{equation*}
