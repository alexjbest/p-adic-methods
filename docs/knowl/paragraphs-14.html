Chabauty-Coleman wrap up.

What if Coleman's bound

\begin{equation*} \# X(\QQ) \le \# X(\FF_p) +2g-2 \end{equation*}

is larger than \(\#X(\QQ)_{\text{known}}\text{.}\) If we carry out Chabauty-Coleman, what can we do if we seem to find “extra” \(p\)-adic points that don't look like they live if \(X(\QQ)\text{?}\)

Try using the Mordell-Weil sieve (developed by Scharashkin in his thesis 99, adapted by Flynn ('04), Poonen-Schaefer-Stoll 07, Bruin-Stoll. Set-up

\begin{equation*} X/\QQ \end{equation*}

a curve of genus \(g \ge 2\text{,}\) \(M \gt 0\) an integer

\begin{equation*} i\colon X\hookrightarrow J \end{equation*}

suppose \(c_M \subseteq J(\QQ)/MJ(\QQ)\) is a set of residue classes for which we want to show that no rational point \(P \in X(\QQ)\) maps to \(c_M\) under \(\pi \circ i\text{.}\)

Simplest case: pick a good prime \(v\)

\begin{equation*} \xymatrix{X(\QQ) \ar[r]\ar[d] & J(\QQ)/MJ(\QQ) \ar[d]\\ X(\FF_v) \ar[r] & J(\FF_v)/MJ(\FF_v)} \end{equation*}

if \(\alpha _v(c_M) \cap \beta _v(X(\FF_v)) = \emptyset\) then done. Typically this is not enough: More generally consider set \(S\) of good primes and the commutative diagram

\begin{equation*} \xymatrix{X(\QQ) \ar[r]\ar[d] & J(\QQ)/MJ(\QQ) \ar[d]\\ \prod_v X(\FF_v) \ar[r] & \prod_v J(\FF_v)/MJ(\FF_v)} \end{equation*}

then it suffices to show that \(\alpha _s(c_M) \cap \beta _s(\prod_v X(\FF_v)) = \emptyset\text{.}\)

The goal is then to find a good set of \(S\) s.t.

\begin{equation*} A(S, c_M) = \left\{c\in c_M : \alpha _S(c) \in \beta _s(\prod_v X(\FF_v))\right\} \end{equation*}

is empty.

Heuristically the size of \(A(S,c_M)\) is as follows: For a good prime

\begin{equation*} X_{M,v} = \beta _v(X(\FF_v)) \end{equation*}
\begin{equation*} \gamma (v,M) = \frac{\# X_{M,v}}{\#J(\FF_v)/ M(J(\FF_v))} \end{equation*}

Note : \(v\) is only useful if \(\gamma (v,M) \lt 1\text{.}\)

Expected size of \(A(S, M)\) is

\begin{equation*} \#c_M \prod_c \gamma (v, M) \end{equation*}

want this to be small.

Difficulties in using the sieve, the set, \(A(S, c_M)\) can be large. The computation of images of \(X(\FF_v)\) in \(J(\FF_v)\) can become infeasible (the computation requires \(v\) discrete logs in \(J(\FF_v)\)). Can be mitigated by using primes for which \(\#J(\FF_v)\) is sufficiently smooth.

Two variations on Chabauty-Coleman: Pass to a collection of covering curves, (difficulty: constructing covers) Elliptic Chabauty method: (difficulty computing \(E(K)\) for \(K/\QQ\) a larger degree number field.

These methods can potentially compute \(X(\QQ)\) when \(\rk J(\QQ) \ge g\text{.}\)

Theorem 2.47. Chevalley-Weil.

Let \(f \colon Y\to X\) be an unramified morphism of curves \(/\QQ\) then there is a computable finite extension \(K/\QQ\) such that

\begin{equation*} f\inv(X(\QQ)) \subseteq Y(K)\text{.} \end{equation*}
Theorem 2.48. Wetherell '97.

There is a finite set of unramified covering curves \(Y_i \to X\) over \(\QQ\) (all isomorphic over \(\overline\QQ\)), such that

\begin{equation*} X(\QQ) \subseteq \bigcup_{i=1}^n f_i(Y_i(\QQ))\text{.} \end{equation*}
Remark 2.49.

When \(X\) is an elliptic curve this is descent in the proof of the Mordell-Weil theorem.
