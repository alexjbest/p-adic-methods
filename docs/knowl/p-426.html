We choose a trivialization

\begin{equation*} s_0 (b, \cdot) \colon (\QQ \oplus V_{\dR} \oplus \QQ(1))\otimes \sheaf O_Y \xrightarrow \sim A_Z(b)|_Y \end{equation*}

s.t. the connection \(\nabla\) on \(A_Z\) via this trivialization is given by

\begin{equation*} s_0\inv \nabla s_0 = \diff + \Lambda \end{equation*}

where

\begin{equation*} \Lambda = \begin{pmatrix} 0 \amp 0 \amp 0 \\ \omega \amp 0 \amp 0 \\ \eta \amp \omega ^\transpose Z \amp 0 \end{pmatrix} \end{equation*}

for some \(\eta\) of the third kind on \(X\) in space spanned by

\begin{equation*} \{ \omega _{2g}, \ldots, \omega _{2g+d-2}\} \end{equation*}
\begin{equation*} \omega = \{ \omega _0 , \ldots, \omega _{2g-1}\}\text{.} \end{equation*}

\(Z\) is the matrix of the Tate class with respect to

\begin{equation*} H^1_{\dR} (X) \otimes H^1_{\dR}(X) \end{equation*}

(associated to the correspondence \(Z\)). Fix a basis

\begin{equation*} \{1,T_0, \ldots, T_{2g-1}, S\} \end{equation*}

to write down \(s_0\text{.}\)
