Algorithm 2.68. Local height at \(p\).

Output: \(h_p(D_1,D_2)\text{.}\)

  1. Let \(\omega \) be a differential in \(T(K)\) with \(\Res(\omega ) = D_1\text{.}\)
  2. Compute \(\Psi (\omega ) = \sum_{i=0}^{2g-1} a_i\omega _i \in H^1_{\dR}(X)\text{.}\) Then
    \begin{equation*} \Psi (\omega ) - \sum_{i=0}^{g-1} a_i\omega _i \in W \end{equation*}
    let \(\omega _{D_1} = \omega - \sum_{i=0}^{g-1} a_i \omega _i\text{.}\)
  3. Compute using the previous algorithm
    \begin{equation*} h_p(D_1,D_2) = \int_{D_2} \omega _{D_1}\text{.} \end{equation*}
