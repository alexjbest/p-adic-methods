Now we can describe the Hodge filtration on \(A_Z\) with respect to our chosen trivialization, \(s_0\) i.e. an explicit isomorphism

\begin{equation*} s^{\Fil} \colon ( (\QQ \oplus V_{\dR} \oplus \QQ(1))\otimes \sheaf O_Y) \xrightarrow \sim A_Z \end{equation*}

that respects the Hodge filtration on both sides, where the LHS is given by

\begin{align*} \Fil^{-1}\amp = (\QQ \oplus V_{\dR} \oplus \QQ(1)) \otimes \sheaf O_Y\\ \Fil^0 \amp = (\QQ \oplus \Fil^0V_{\dR}) \otimes \sheaf O_Y\\ \Fil^1 \amp = 0\text{.} \end{align*}
