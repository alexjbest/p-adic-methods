Let \(z = z_\infty \) be a uniformizer at \(x\)

\begin{equation*} \partial _t z = 1 \end{equation*}

any function

\begin{equation*} f \in K(X)^\times \end{equation*}

then has a well-defined “value” at x

\begin{equation*} f[x] = \frac{f}{z^m}(x) \end{equation*}

where \(m = \ord_x(f)\text{.}\) This depends only on \(t\) and not \(z\text{.}\) To do this for local p-adic heights use Besser's p-adic Arakelov theory, JNT 2005.
