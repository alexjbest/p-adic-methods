Definition 2.98. Mixed extensions.

We define \(M_{\Fil,\phi } (\QQ_p, V_\dR, \QQ_p(1))\) to be the category of mixed extensions of filtered \(\phi \)-modules, the objects are triples \((M, M_{\bullet}, \psi _\bullet)\) where

  1. \(M\) is a filtered \(\phi \)-module.
  2. \(M_{\bullet}\) is a filtration by sub-filtered \(\phi \)-modules
    \begin{equation*} M = M_0 \supseteq M_1 \supseteq M_2 \supseteq M_3= 0\text{.} \end{equation*}
  3. \(\psi _\bullet\) are isomorphisms of filtered \(\phi \)-modules
    \begin{align*} \phi _0 \amp\colon M_0/M_1\xrightarrow \sim \QQ_p\\ \phi _1 \amp\colon M_1/M_2\xrightarrow \sim V_\dR\\ \phi _2 \amp\colon M_2/M_3\xrightarrow \sim \QQ_p(1) \end{align*}

and the morphisms of this category are morphisms of filtered \(\phi \)-modules which also respect the filtrations \(M_\bullet\) and commute with the isomorphisms \(\psi _i\) and \(\psi _i'\text{.}\)

Let

\begin{equation*} M_{\Fil, \phi }(\QQ_p, V_{\dR}, \QQ_p(1)) \end{equation*}

denote the set of isomorphism classes of objects in this category.
