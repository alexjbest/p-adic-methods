Let \(X/\QQ\) be a smooth projective curve of genus \(g\text{.}\) Suppose \(\rk J(\QQ) = g\) and that

\begin{equation*} \overline{J(\QQ)} \subseteq J(\QQ_p) \end{equation*}

has finite index. A quadratic Chabauty pair \((\theta , S)\) is a function

\begin{equation*} \theta \colon X(\QQ_p) \to \QQ_p \end{equation*}

and a finite set \(S\) such that

  1. On each residue disk of \(X(\QQ_p)\) the map
    \begin{equation*} (\operatorname{AJ}_b, \theta ) \colon X(\QQ_p) \to H^0(X, \Omega ^1)^* \times \QQ_p \end{equation*}
    has Zariski dense image and is given by a convergent \(p\)-adic power series.
  2. There exists an endomorphism \(E\) of
    \begin{equation*} H^0(X_{\QQ_p}, \Omega ^1)^*\text{,} \end{equation*}
    a functional \(c\in H^0(X_{\QQ_p}, \Omega ^1)^*\) and a bilinear form \(B\)
    \begin{equation*} B\colon H^0(X_{\QQ_p}, \Omega ^1)^* \otimes H^0(X_{\QQ_p}, \Omega ^1)^* \to \QQ_p \end{equation*}
    such that for all \(x \in X(\QQ)\text{,}\) we have
    \begin{equation} \theta (x) - B(\operatorname{AJ}_b(x), E(\operatorname{AJ}_b(x)) + c)\in S\text{.}\tag{2.3} \end{equation}
