The connections on

\begin{equation*} A_Z^\dR(b)^\an|_Y \end{equation*}

and

\begin{equation*} \phi ^* A_Z^\dR(b)^\an|_Y \end{equation*}

are described with respect to \(s_0\) and are equal to \(\diff + \Lambda \) (as before) and \(\diff +\Lambda _\phi \text{,}\) where

\begin{equation*} \Lambda _\phi = -\begin{pmatrix} 0 \amp 0 \amp 0 \\ \phi ^* \omega \amp 0\amp 0 \\ \phi ^* \eta \amp \phi ^* \omega^* Z\amp 0\end{pmatrix} \end{equation*}

so to make the frobenius structure explicit we must compute \(G\) s.t.

\begin{equation*} \Lambda _\phi G + \diff G = G\Lambda \end{equation*}

where \(G = \Phi \inv _Z\) (inverse of frobenius structure).
