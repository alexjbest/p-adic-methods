We will begin with \(p\)-adic heights on elliptic curves. Let \(E/\QQ\) be an elliptic curve, then a \(p\)-adic height \(h\) is a function

\begin{equation*} h\colon E(\overline \QQ) \to \QQ_p \end{equation*}

that plays a similar role to the canonical height

\begin{equation*} \hat h E(\overline \QQ) \to \RR\text{.} \end{equation*}
