\(X\text{:}\) Smooth projective curve \(\FF_q\text{,}\) birational to

\begin{equation*} Q(x,y) = y^{d_x -1} + Q_{d_x - 1}y^{d_x-1} + \cdots + Q_0 = 0 \end{equation*}

is irreducible, where \(Q_i \in \FF_q\lb x \rb\) for \(i = 0 , \ldots, d_x -1\text{.}\) This is part II of Tuitman [93] as we have a not necessarily smooth model.
