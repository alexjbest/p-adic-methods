Algorithm 2.66. Coleman integral of differential of the third kind, with poles in non-weierstrass disks.

Input: \(\omega \) with \(\Res(\omega ) = (P) - (Q)\text{.}\) \(P,Q\in X(\QQ)\) non-weierstrass. \(R,S\in X(\QQ_p)\text{,}\) \(R,S\not\in \operatorname{disk}(P),\operatorname{disk}(Q)\text{.}\) Output:

\begin{equation*} \int_S^R \omega \end{equation*}
  1. Compute \(\Psi (\omega ) \in H^1_{\dR}(X)\text{.}\) Let \(\phi \) be a lift of Frobenius. Let \(\alpha = \phi ^* \omega - p\omega \text{.}\) Use \(\Psi (\omega )\) and frobenius equivariance. We have
    \begin{equation*} \Psi (\alpha ) = \Psi (\phi ^* \omega - p\omega ) \end{equation*}
    \begin{equation*} = \Psi (\phi ^* \omega) -\Psi ( p\omega ) \end{equation*}
    \begin{equation*} = \phi ^* \Psi (\omega) -p\Psi (\omega ) \end{equation*}
  2. Let \(\beta \) be a 1-form with \(\Res(\beta ) = (R)- (S)\text{.}\) Compute \(\Psi (\beta )\text{.}\)
  3. Compute \(\Psi (\alpha ) \cup \Psi (\beta )\text{,}\) easy since both are elements in \(H^1_{\dR}\) that we just computed.
  4. Compute
    \begin{equation*} \int_{\phi (S)}^S \omega \end{equation*}
    and
    \begin{equation*} \int_R^{\phi (R)} \omega \end{equation*}
    tiny
  5. Compute
    \begin{equation*} \sum_A \Res_A (\alpha \int \beta )\text{.} \end{equation*}
  6. \begin{equation*} \int_S^R \omega = \frac{1}{1-p} \left(\Psi (\alpha ) \cup \Psi (\beta ) + \sum_A \Res_A(\alpha \int \beta ) - \int_{\phi (S)}^S \omega - \int_R^{\phi (R)} \omega \right) \end{equation*}
