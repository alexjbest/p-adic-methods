Theorem 2.70. Quadratic Chabauty for integral points on hyperelliptic curves B.-Besser-Muller.

Let \(f(x) \in \ZZ\lb x\rb \) be monic separable polynomial of degree \(2g+1 \ge 3\text{,}\) that does not reduce to a square modulo \(q\) for any prime \(q\text{.}\) (in the paper monic is not used, this condition then restricts the reduction type) Let \(U = \Spec (\ZZ\lb x,y\rb /(y^2 - f(x)))\) and let \(X\) be the normalization of the proj closure of the generic fiber of \(U\text{.}\) Let \(J\) be the Jacobian on \(X\) and assume \(\rk J(\QQ) = g\text{,}\) choose a prime \(p\) of good ordinary reduction. Suppose that the \(p\)-adic coleman integrals

\begin{equation*} f_i(z) = \int_\infty ^z \omega _i = \int_\infty ^z \frac{x^i \diff x}{2y} \end{equation*}

then there exists explicitly computable constants \(\alpha _{ij} \in \QQ_p\) s.t. the locally analytic function

\begin{equation*} \rho (z) = \theta(z) - \sum_{0\le i \le j \le g-1 } \alpha _{ij} f_i(z) f_j(z)\text, \end{equation*}

where \(\theta(z) = h_p((z) - (\infty ),(z) - (\infty ))\) is an extension of the Coleman-Gross local height at \(p\) which takes values in

\begin{equation*} \mathcal U(\ZZ\lb 1/p\rb ) \end{equation*}

in an effectively computable finite set \(S\subseteq \QQ_p\text{.}\)
