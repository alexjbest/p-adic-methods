Roughly speaking intersections are computing denominators which is why its not obvious how to go beyond integral points using this construction. e.g. for elliptic curves have Mazur-Stein-Tate \(p\)-adic height.

\begin{equation*} h(P) = \frac 1p \log_p(\sigma (P)) - \frac 1p \log_p(D(P)) \end{equation*}

Coleman-Gross

\begin{equation*} h(P -\infty ) = h_p(P-\infty ) + \sum_{v\ne p} h_v(P-\infty ) \end{equation*}

extended appropriately.
