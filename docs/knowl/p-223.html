e.g. if

\begin{equation*} X\colon y^2 = f(x) \end{equation*}

is a hyperelliptic curve

\begin{equation*} D_1 = (P) - (Q),\,P,Q\in X(\QQ),\,y(P),y(Q) \ne 0 \end{equation*}

can we construct \(\omega \) with \(\Res(\omega ) = D_1\text{?}\) We want residue 1 at \(P\) and -1 at \(Q\text{.}\) So \(\omega \) has simple poles at \(P,Q\text{.}\) e.g.

\begin{equation*} \omega = \frac{\diff x}{2y}\left(\frac{y+y(P)}{x-x(P)} - \frac{y + y(Q)}{x-x(Q)}\right) \end{equation*}

has \(\Res(\omega ) = D_1\text{.}\) But there are lots more!
