Definition 2.81.

Let

\begin{equation*} G^{\lb 1\rb } = G \end{equation*}
\begin{equation*} G^{\lb 2\rb } =\lb G^{\lb 1\rb } ,G\rb \end{equation*}
\begin{equation*} \ldots \end{equation*}
\begin{equation*} G^{\lb i+1\rb } =\lb G^{\lb i\rb } ,G\rb \end{equation*}

then

\begin{equation*} G^{\lb 1\rb } \unrhd G^{\lb 2\rb } \unrhd\cdots \end{equation*}

is a descending chain of normal subgroups of \(G\) called the lower central series of \(G\text{.}\) Let \(G_n = G/G^{\lb n + 1\rb }\text{.}\)
