To compute \(h(Q)\) for arbitrary \(Q\in E(\QQ)\text{.}\) Let \(n_1 = \#E(\FF_p)\text{,}\) \(n_2 = \lcm_v(\{c_v\})\text{,}\) let \(n = \lcm(n_1,n_2)\) then \(P=nQ\) satisfies 1) and 2) from earlier. So compute \(h(P) = h(nQ)\) and then

\begin{equation*} h(Q) = \frac{1}{n^2 } h(nQ) = \frac{1}{n^2} h(P)\text{.} \end{equation*}
