The iterated Coleman integral

\begin{equation*} \int \omega _n\omega _{n-1} \omega _1 \end{equation*}

is the \(y_n\) coordinate of a solution to the following system of differential equations:

\begin{equation*} \diff y = \Omega y,\,\Omega = \begin{pmatrix} 0 \amp 0 \amp \cdots \amp 0 \\ \omega _1 \amp 0 \amp \cdots \amp 0\\0\amp \omega _2 \amp \cdots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp \vdots \\ 0 \amp 0 \amp \cdots \amp 0\\ 0 \amp 0 \amp \cdots \omega _n \amp 0 \end{pmatrix}\text{.} \end{equation*}

With \(y_0 = 1\text{.}\) This is a unipotent differential equation

\begin{equation*} y= \begin{pmatrix} y_0\\\vdots\\y_n \end{pmatrix}\text{.} \end{equation*}
