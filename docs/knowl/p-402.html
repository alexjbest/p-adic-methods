A natural analogue of Nekovář's construction is to start with the input of a cohomology class \(\xi \) in \(H^1(G_T, \QQ_p)\) and define at all bad primes \(v\) an algebraic function

\begin{equation*} H^1(G_{\QQ_v}, U) \to \QQ_p \end{equation*}

which when restricted to

\begin{equation*} H^1(G_{\QQ_v}, \QQ_p(1)) \end{equation*}

is the cup product with \(\xi \text{.}\) Given a splitting of the Hodge filtration can do this. But to get equality for Selmer varieties better to have a construction with linearity properties analogous to the global height pairing. So here we describe how to embed \(\Sel(U)\) into

\begin{equation*} H_{st}^1(G_T, \ldots) \subseteq H^1(G_T, \ldots) \end{equation*}

via twisting. Then apply Nekovář's construction giving local functions from

\begin{equation*} \Sel(U) \to \QQ_p\text{.} \end{equation*}
