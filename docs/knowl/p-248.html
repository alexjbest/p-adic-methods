Then look at \(h = h_p + \sum_{v \ne p} h_v\) note

  1. \(h\) is a quadratic form, so can be written in terms of a basis of space of quadratic forms for \(J(\QQ)\) and this can be done using Coleman integrals.
    \begin{equation*} h(z-\infty , z-\infty ) = \sum \alpha _{ij} f_i(z) f_j(z)\text{,} \end{equation*}
  2. \(h_p\) is a locally analytic function and in the extension to self-pairing:
    \begin{equation*} h_p(z-\infty , z-\infty ) = -2\sum_{i=0}^{g-1} \int_\infty ^z \omega _i\bar \omega _i,\,\omega _i = \frac{x^i \diff x}{2y},\,\bar\omega _i\text{ cup product duals}\text{.} \end{equation*}
  3. The sum
    \begin{equation*} \sum_{v\ne p} h_v(z-\infty ,z-\infty ) \end{equation*}
    takes on finitely many values in \(S\) when restricted to \(p\)-integral points. The set \(S\) can be computed by knowing the reduction of \(X\) at bad primes.
  4. Then rewrite
    \begin{equation*} h- h_p \in \{ \sum_{v\ne p} h_v \} = S \end{equation*}
  5. this \(\rho \) can be computed as a convergent power series in each residue disk. So now pretend we are working in classical chabauty Coleman. Expand \(\rho \) in each disk, set equal to each value in \(S\) solve for all \(t \in U(\ZZ_p)\) s.t. \(\rho (z) \in Z\text{.}\) Take all such points call that \(X(\ZZ_p)_2\text{.}\)
  6. It's possible that \(X(\ZZ_p)_2\) is strictly larger than the known points in \(U(\ZZ)\text{.}\) In this case run 1-4 on a collection of good ordinary \(p\) and run Mordell-Weil sieve.
