By our formula

\begin{equation*} h_p(A_Z(b,x)) = \chi _p(\gamma _\phi -\gamma _{\Fil} - \beta _\phi ^\transpose \cdot s_1(\alpha _\phi ) - \beta _{\Fil}^\transpose s_2(\alpha _\phi ))\text{,} \end{equation*}

we see that we want to compute an explicit description of \(A_Z(b,x)\) as a filtered \(\phi \)-module. This we do in two parts: The Hodge filtration (today) and Frobenius structure (via solving a \(p\)-adic differential equation using Tuitman's algorithm, see later), of certain universal objects \(A_Z^\dR\text{.}\)
