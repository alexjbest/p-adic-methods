Finite pole order reduction:.

For \(i = 0, \ldots, 2g-1\text{,}\) find \(f_{i,0} \in \mathcal R^\dagger \otimes \QQ_p\) such that

\begin{equation*} \phi^\ast \omega_i = df_{i,0} + G_i \frac{dx}{r} \end{equation*}

where \(G_i \in \mathcal R \otimes \QQ_p\) only has poles above \(\infty\text{.}\)
